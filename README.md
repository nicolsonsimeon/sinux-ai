# Sinux AI

Plateforme d'intelligence artificielle multimodale.

## Objectifs

Sinux AI sera capable de :

- Conversation IA
- Mémoire intelligente
- Génération d'images
- Génération vidéo
- Reconnaissance vocale
- Synthèse vocale
- Agents intelligents

## Architecture

Frontend : Flutter

Backend : FastAPI

Database : PostgreSQL

AI Engine : LLM + Agents
